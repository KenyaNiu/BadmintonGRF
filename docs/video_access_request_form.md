## BadmintonGRF Video Access Request Form

### 1) Applicant information
- **Name**
- **Email**
- **Institution**
- **Lab / Department**
- **PI / Supervisor (name + email)**
- **Website (optional)**

### 2) Project information
- **Project title**
- **Short project description (5–10 sentences)**
- **How raw videos are required (why processed data is insufficient)**
- **Expected outputs** (paper, thesis, open-source code, etc.)

### 3) Ethics / compliance
- **Do you have IRB/ethics approval or equivalent clearance?**
  - If yes: provide the approval ID / document (or a short statement if confidential).
  - If no: explain why not required under your institution’s policy.

### 4) Data security plan
- **Storage location** (server / encrypted drive / secure cloud bucket)
- **Access control** (who will have access; how access is granted/revoked)
- **Retention period** (how long you will keep the videos)
- **Deletion plan** (how you will delete data upon request or project end)

### 5) Agreement
By submitting this form, you agree to comply with the policy in:
- `docs/video_access_policy.md`

### 6) Signature
- **Name**
- **Date**
- **Role** (PI / student / researcher)
